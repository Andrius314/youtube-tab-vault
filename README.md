# YouTube Tab Vault

Naujas atskiras projektas YouTube nuorodu valdymui su kategorijomis ir aprasymais.

## Paleidimas

1. Atidaryk `youtube-tab-vault/index.html` narsykleje.
2. Pridek YouTube nuoroda.
3. Jei nori, spausk `Auto paimti info is URL`.
4. Priskirk kategorija ir aprasyma.

## Pagrindines funkcijos

- YouTube stiliaus isdestymas (top bar + kairys kategoriju menu + video korteles).
- Kategorijos ir filtravimas.
- Korteles su thumbnail, kanalu ir aprasymu.
- Paieska ir rusiavimas.
- Redagavimas, trynimas, perkelimas i kita kategorija.
- JSON eksportas ir importas.
- Duomenys saugomi `localStorage`.
